# WakeSaga GPT-image-2 Onboarding Language Exploration

Goal: establish a stronger onboarding setup language after the prior Trip/BFF-style implementation missed the mark.

Product direction:
- WakeSaga turns alarm setup into the start of a daily anime training arc.
- The onboarding should still feel viral, social, and Gen Z, but not like a generic friend-trip app.
- Anime identity is core: episode framing, morning rival, training arc, expressive stickers, clean manga energy.
- Avoid dark RPG HUDs, cyberpunk dashboards, tiny paragraphs, generic SaaS gradients, and copied Trip/BFF branding.

Concept set:
1. Group Chat Pilot
2. BFF Quiz Cards
3. Morning Persona Builder
4. Wake Receipt Setup

Use these as visual-language boards, not final copy. Generated text may need cleanup before implementation.
