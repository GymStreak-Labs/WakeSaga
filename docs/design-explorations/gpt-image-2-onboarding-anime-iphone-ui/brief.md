# WakeSaga GPT-image-2 Anime iPhone Onboarding Exploration

## Product Brief

WakeSaga is an AI alarm and morning motivation app that turns waking up into the first episode of a user's daily training arc. This exploration extends the cleaner anime/iPhone product UI into first-run onboarding.

## Design Direction

**Clean Anime Onboarding**: a native-feeling iPhone onboarding flow with anime identity as the emotional layer. Each screen should have one clear job, one main illustration/card, and one obvious CTA.

## Generated Anchor Screens

1. `generated/01-cold-open.png` - the core promise: tomorrow becomes Episode 001.
2. `generated/02-morning-enemy.png` - diagnose the user's morning obstacle.
3. `generated/03-saga-loop.png` - teach the alarm → wake gate → episode loop.
4. `generated/04-choose-arc.png` - choose the user's daily training arc.
5. `generated/05-choose-gate.png` - choose the wake-up action gate.
6. `generated/06-plan-reveal.png` - reveal tomorrow's opening scene.

## Visual Notes

- Use a tall iPhone-like screen ratio with safe-area spacing.
- Avoid dense manga panels, overloaded badges, and tiny labels.
- Use soft cream backgrounds, black typography, coral CTA, cyan/gold accents.
- Keep anime identity through original character art, episode language, rival/mentor framing, and collectible card surfaces.
- UI text should be clean, large, and sparse.
