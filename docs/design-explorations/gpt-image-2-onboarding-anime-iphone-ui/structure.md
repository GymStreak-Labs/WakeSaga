# Clean Anime iPhone Onboarding Structure

## Spine

The onboarding should still be long-form, but each screen should feel like a simple iPhone app screen, not a poster or dense quiz. The flow is:

`promise → diagnose → reframe → personalize → configure Wake Quest → commit → reveal plan`

## Full Flow

1. Cold open: tomorrow becomes `Episode 001`.
2. Promise: wake up before the day starts without you.
3. Question: what usually beats you after the alarm?
4. Question: what steals the first 10 minutes?
5. Education: sleep inertia is biology, not weakness.
6. Old loop: alarm → snooze → scroll → panic.
7. Saga loop: alarm → Wake Quest → Morning Episode → first move.
8. Choose tomorrow's arc: Study, Gym, Deep Work, Comeback, Monk Mode, Recovery.
9. Choose mission focus.
10. Choose rival: warm bed, phone spiral, low mood, late-start panic.
11. Choose Wake Quest: object hunt, sky photo, make bed, pushups, water check, desk photo, shoes on, spoken vow.
12. Explain selected quest in one simple sentence.
13. Set wake time.
14. Choose wake days.
15. Choose narrator: Mentor, Rival, Future Me, Hype Friend.
16. Choose tone: calm, sharp, cinematic, recovery.
17. Preview wake jolt.
18. Permission primer for notifications/alarm behavior.
19. Commitment: sign tomorrow's opening scene.
20. Rendering screen: building wake jolt, Wake Quest, episode, wake card.
21. Plan reveal: alarm time, Wake Quest, narrator, mission.
22. Save account.
23. Trial/paywall after the plan feels real.

## Anchor Screens To Implement First

1. **Cold Open** - one anime sunrise hero, one headline, one CTA.
2. **Morning Enemy** - big question, four simple cards.
3. **Saga Loop** - three-step visual diagram, no paragraph.
4. **Choose Arc** - six large arc cards with icons/crest art.
5. **Choose Wake Quest** - selectable wake-action mission cards.
6. **Plan Reveal** - tomorrow's opening scene card with alarm time and CTA.

## Implementation Rule

Keep each screen to one primary decision. If the Wayk-length flow needs more screens, split them instead of making one screen dense.

## Naming Update

Use **Wake Quest** as the mechanic name, not `Wake Gate`. Individual choices are quest missions inspired by Wayk-style dismissal tasks. Good mission labels:

- Object Hunt
- Sky Photo
- Make Bed
- Pushups
- Water Check
- Desk Photo
- Shoes On
- Spoken Vow

Use `Wake Mission` only as supporting language inside a quest card, not as the top-level feature name.
