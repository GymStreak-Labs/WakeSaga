# WakeSaga App Icon Concepts — Face Close-up + Dawn/Sunlight

Generated directly with Nano Banana (`fal-ai/nano-banana-pro`) for concept exploration only. This pass does **not** use AppStoreCopilot saved styles or project state. No live app icon files were changed.

## Families

- `01-06`: super close-up determined anime face / eye variants. These are intentionally cropped large so they can survive icon scale.
- `07-12`: anime dawn / first-light variants. These aim for simpler alarm + sunrise silhouettes visible from far away.

## Judging Criteria

- One clear silhouette at 44 px.
- No text, letters, numbers, UI, scenes, or tiny detail.
- Critical subject inside iOS safe area.
- Fits WakeSaga's Cold Open alarm identity.
